# ReadMe for the Pigeon Breeder
## Info
Python Version: 3.10

Most recent version always in the developer branch  
Most recent stable version in the release branch

## How To Use
Open the CMD  
Navigate to the code folder of this project  
Run 'python3.10 run.py'  
Enjoy!
